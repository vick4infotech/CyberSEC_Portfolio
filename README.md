# Victor Ayenikanju Cybersecurity Portfolio

A React + Vite cybersecurity portfolio using Tailwind CSS, Framer Motion, React Router, and a Linux-inspired dashboard interface.

## Run locally

```bash
npm install
npm run dev
```

Open the exact URL printed by Vite, usually:

```text
http://localhost:5173/
```

Do not open `index.html` directly. Vite must serve the app.

## Blank page fix included

This version pins stable dependency versions instead of using `latest`, uses `HashRouter`, and includes an on-page error fallback. If React fails before rendering, the page will show the exact JavaScript error instead of staying blank.

## Build

```bash
npm run build
npm run preview
```
